What is the Document of Object Model (DOM)?

The Document Object Model is a programming interface for web documents. It represents the page so that programs can change the document structure, style and content.

How do you select an element from DOM?

To select an element from the DOM in JavaScript, you can use one of the following methods:

1. By Id : document.getElementById('elementId');
1. By Id : document.getElementById('elementId');
2. By Class : document.getElementsById('className');
3. By Tag : document.getElementsById('tagName');
4. By CSS Selector : a. document.querySelector('selector') //For single elements
b. document.querySelectorAll('selector') //For multiple elements

What is event delegation in the context of the DOM and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of individual child elements. This listener catches events from child elements as they "bubble up" through the DOM (event bubbling).

Why it's useful :
1. Performance: Reduces memory use by avoiding multiple event listeners.
2. Dynamic elements: Handles events for elements added to the DOM later.
3. Simplified code: Manages events in one place, even for many child elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can use the following methods:

1. Attributes :
a.Get an attribute : element.getAttribute('attributeName');
a.Set an attribute : element.setAttribute('attributeName','value');
a.Remove an attribute : element.removeAttribute('attributeName');

2. Styles :
a. Directly modify styles : element.style.property = 'value';
//Example : element.style.color = 'red';
b. Modify multiple styles with : element.style.cssText = 'color: red; background-color: yellow;';
c. Add or remove classes : element.classList.add('className');
element.classList.remove('className');
element.classList.toggle('className');
